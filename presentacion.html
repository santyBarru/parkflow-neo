Paso 0 — Antes de empezar

Levantar el sistema

Todo corre localmente. Abre dos terminales y ejecuta estos comandos.

Terminal 1 — Backend
#Agregar PostgreSQL al PATH
$ $env:PATH += ";C:\Program Files\PostgreSQL\17\bin"
#Ir al backend
$ cd Back-end/backend
$ ./mvnw spring-boot:run
→ Started on port 8080 ✓
→ http://localhost:8080/actuator/health
Terminal 2 — Frontend
#Ir al frontend
$ cd parkflow-frontend
$ npm run dev
→ Local: http://localhost:3000 ✓

Seguridad integrada

DevSecOps

Seguridad no es un paso final — está integrada desde el primer commit.

🔑
Gitleaks — Secrets Scanning
Escanea cada commit buscando tokens, contraseñas y secretos expuestos accidentalmente en el código.
En pipeline
🔬
Trivy — Container Scanning
Escanea el sistema de archivos y las imágenes Docker buscando CVEs y vulnerabilidades conocidas.
En pipeline
⚡
Semgrep — SAST
Análisis estático del código fuente buscando patrones inseguros del OWASP Top 10.
En pipeline
🛡️
OWASP + JWT + BCrypt
Dependency check de librerías, autenticación con tokens firmados HS256 y contraseñas hasheadas.
